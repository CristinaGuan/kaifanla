# kaifanla
Angular 开饭啦项目

运行：
1、前提：需要安装XAMPP工具，启动mysql和Apache，下载地址：https://www.apachefriends.org/zh_cn/download.html
        下载phpMyAdmin 下载地址：https://www.phpmyadmin.net/
        
2、登录phpMyAdmin,导入kaifala数据库，查看kaifanla.sql

3、访问http://localhost/kaifanla/kfl_Index.html#/home
